# DATA 37100 — Final Project Package

This folder contains materials for the course final project.

Projects are designed for a **~20-hour** scope and emphasize:

- small, fast baselines
- controlled experiments (two‑knob studies)
- failure mode identification
- clear, evidence-based interpretation

## Start here

1. Read `FINAL_PROJECT.md`
2. Review `DATASETS.md`
3. Submit a short proposal using `PROJECT_PROPOSAL.md`
4. Use `final/starter/` if you want a quick, reproducible baseline setup

## What models are supported?

You must use **at least two** model families:

- GAN (DCGAN) — Week 4
- Transformer — Week 5
- Diffusion — Week 7
